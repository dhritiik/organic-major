# OrganicFlow ⚛️

**The Fluid Knowledge Graph of Molecular Discovery.**

OrganicFlow is an immersive, interactive knowledge graph designed to visualize the vast and intricate world of chemistry. By mapping the relationships between elements, backbones, and molecules, it turns complex data into a seamless, explorable flow.

## 🌟 Features

-   **Deep Elemental Roots:** Connects every molecule to its core atoms (Hydrogen, Oxygen, Nitrogen, etc.).
-   **Carbon-Backbone Spine:** A clear hierarchical structure from simple methane to complex decane chains.
-   **Expansive Knowledge Base:** Procedurally generates homologous series, functional groups, and aromatics.
-   **Reaction Intelligence:** Visualizes the pathways of organic reactions (Oxidation, Substitution, Esterification).
-   **Mechanism Playback:** Watch step-by-step animations of the electron flows and bond changes in key reactions.
-   **Sleek Visualization:** A high-performance grid layout with zero-overlap and glassmorphic UI.

## 🚀 Getting Started

1.  **Install Dependencies:**
    ```bash
    npm install
    ```
2.  **Run Development Server:**
    ```bash
    npm run dev
    ```
3.  **Build for Production:**
    ```bash
    npm run build
    ```

## 🛠️ Tech Stack

-   **Framework:** React + Vite
-   **Graphing:** React Flow
-   **Motion:** Framer Motion
-   **Styles:** TailwindCSS
-   **Icons:** Lucide React
